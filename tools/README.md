# Tools Folder

This folder contains scripts and helpers to manage and test PostgreSQL projects.

## Scripts

- `import_csv.sh` → Import all CSV datasets into the database
- `export_table.sh` → Export a table to CSV
- `generate_fake_data.py` → Generate fake datasets for testing
- `run_query.sh` → Execute a SQL file on a database
- `backup_db.sh` → Backup a PostgreSQL database
- `restore_db.sh` → Restore a PostgreSQL database
- `analyze_performance.sql` → Run EXPLAIN ANALYZE on important queries
